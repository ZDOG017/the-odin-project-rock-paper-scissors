Rock, Paper, Scissors

Welcome! This is a simple Rock, Paper, Scissors game you can play against the computer. The game uses browser prompts for your choices and prints results to the console.

How to play

  1. Open this page in your browser.
  2. Open the browser console (right-click → Inspect → Console) to see round-by-round results and the final score.
  3. When prompted, type Rock, Paper, or Scissors. Case does not matter; leading/trailing spaces are ignored.
  4. The computer will choose randomly each round. Results and the running score are printed in the console.
  5. After the chosen number of rounds, the final score and winner will be shown in the console.

Start the game

The game may start automatically depending on the version of the script in scripts.js. If it does not, you can start it manually by opening the console and running:

playGame(5)

Replace 5 with another number to play a different number of rounds, for example playGame(3).
